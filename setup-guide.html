RepCommish — Developer Setup

Mac onboarding for joining the project

You only need to download two GUI apps the normal way — a terminal and a code editor. Everything else gets installed by typing commands into the terminal. Total setup time is roughly 30 minutes including downloads.

1Download two apps

Warp (terminal)
Where you'll run every command in this guide. The free tier is fine.
warp.dev — or use the Terminal app that's already built into your Mac (Applications → Utilities → Terminal).
Cursor or VS Code (code editor)
Where you'll edit code. Cursor is VS Code with AI built in — either works.
cursor.com  or  code.visualstudio.com

2Install Homebrew

Homebrew is the package manager for Mac. It's how you install Node, Git, and other dev tools. Open your terminal and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

It'll ask for your Mac password (the same one you use to log into your computer) and take a few minutes. When it finishes, it may print a couple of extra echo commands telling you to add Homebrew to your PATH — copy/paste and run those if it does.

Verify it worked:

brew --version

Should print something like Homebrew 4.x.x.

3Install Node, Git, Python

One command:

brew install node git python3

Verify each one:

node -v      # should be v18 or higher
git --version
python3 --version

4Get added to the GitHub repo

Julie will add you as a collaborator on github.com/juliestromwall/nitro. You'll get an email invite from GitHub — accept it.

One-time auth: The first time you push code, GitHub will need to know it's you. Easiest path is to install GitHub's CLI and log in once: 
brew install gh
gh auth login
Pick GitHub.com, HTTPS, and "Login with a web browser" when prompted.

5Clone the repo and install dependencies

mkdir -p ~/Projects
cd ~/Projects
git clone https://github.com/juliestromwall/nitro.git
cd nitro
npm install

npm install downloads all the project's JavaScript dependencies. Takes 1–2 minutes.

6Get the .env file from Julie

The project needs Supabase API keys to connect to the database. These live in a file called .env that is intentionally not in GitHub (it's secret).

Do not share this file over email, Slack, or text. Those leak. Julie will send it via 1Password, Bitwarden Send, Signal, or AirDrop if you're nearby.

Once you have the file, put it at:

~/Projects/nitro/.env

Same folder as package.json. The filename starts with a dot, so it'll be hidden in Finder by default — press ⌘⇧. in Finder to show hidden files.

7Run the dev server

cd ~/Projects/nitro
npm run dev

Vite will print a URL like http://localhost:5173/. Open it in your browser.

8Install Playwright (for screenshot-driven AI dev)

When Claude assists you, it'll often want to navigate the local site and take screenshots to verify its work. Playwright is the tool it uses.

python3 -m pip install --user playwright openpyxl
python3 -m playwright install chromium

The Chromium binary is about 170 MB. Playwright drives its own browser — you don't need to install Chrome.

Smoke test

Confirm it works:

python3 -c "
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
    b = p.chromium.launch(headless=True)
    page = b.new_page()
    page.goto('https://example.com')
    page.screenshot(path='/tmp/test.png')
    print('OK')
    b.close()
"
open /tmp/test.png

If OK prints and a screenshot of example.com opens, you're done with this step.

9Install Claude Code (optional)

This is the AI pair-programmer Julie uses. It runs in the terminal, reads the project, and can edit files for you. Same workflow she's been using.

brew install --cask claude-code

Or via npm:

npm install -g @anthropic-ai/claude-code

Then from the project folder:

cd ~/Projects/nitro
claude

It'll prompt you to sign in via claude.ai or supply an API key. It picks up the project's CLAUDE.md rules automatically — same playbook Julie works from.
